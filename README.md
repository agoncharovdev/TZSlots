# TZSlots

## Запуск проекта

```
npm install
npm run dev
```

## Структура проекта

```
public/assets/        # ассеты проекта (графика, звуки и т.д.)
src/main/             # точка входа в приложение
src/base/             # базовые классы, не участвующие в бизнес-логике  
                      # используются для отображения UI
src/services/         # сервисы и вспомогательные хелперы
src/screens/          # пример переключения экранов  
                      # содержит два экрана: `menu` и `slots`

src/screens/slots/    # экран "Slots"  
├── SlotsScreen.ts            # основной класс экрана  
├── controller/               # контроллер экрана  
├── model/                    # модели: конфиг и стейт слотов  
├── views/                    # вьюшки и UI-компоненты  
└── factory/                  # фабрика для создания графики

```
